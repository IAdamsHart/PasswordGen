# Overview
This program will generate a customizable password.
The password has options for:
1. Length
2. Uppercase letters
3. Numbers
4. Special Characters

# Development Environment
* Visual Studio Code  
* Python 3.8.5

# Execution
Set up the program in VSCode  
Execution of the program: `python passwordgen.py`
